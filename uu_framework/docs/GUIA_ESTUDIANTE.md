# Guia del Estudiante

Guia para navegar y usar el sitio web del curso.

## Acceso al Sitio

El sitio del curso esta disponible en:
- **URL**: https://sonder.art/ia_p26/

## Navegacion

### Pagina Principal

La pagina principal muestra el indice general del curso con todos los capitulos y secciones ordenados.

### Menu de Navegacion

En la parte superior encontraras:
- **Inicio**: Volver a la pagina principal
- **Tareas**: Lista de todas las tareas del curso
- **Examenes**: Informacion sobre examenes
- **Proyectos**: Proyectos del curso

### Barra Lateral

En paginas de contenido veras una barra lateral con:
- Navegacion jerarquica del capitulo actual
- Links a secciones relacionadas

### Migas de Pan (Breadcrumbs)

Arriba del contenido veras la ruta de navegacion:
```
Inicio > Capitulo 1 > Seccion 1.1 > Tema actual
```

## Cambiar Tema

### Tema Oscuro / Claro

Haz clic en el icono de luna/sol en la esquina superior derecha para cambiar entre:
- **Eva Unit-01**: Tema oscuro (purpura/verde)
- **Light**: Tema claro

Tu preferencia se guarda automaticamente.

### Fuente para Dislexia

Haz clic en el boton "Aa" para activar/desactivar la fuente OpenDyslexic, disenada para facilitar la lectura a personas con dislexia.

## Tipos de Contenido

### Tareas

Las tareas aparecen con un borde naranja:

```
[TAREA]
Titulo de la tarea
Fecha limite: 1 de febrero de 2026
Instrucciones...
```

### Ejercicios

Los ejercicios de practica tienen borde cyan:

```
[EJERCICIO]
Titulo del ejercicio
Pasos a seguir...
```

### Prompts para LLM

Los prompts para usar con ChatGPT/Claude/Cursor tienen borde purpura y un boton de copiar:

```
[PROMPT]
Texto del prompt...
[Copiar]
```

Haz clic en [Copiar] para copiar el texto al portapapeles.

### Ejemplos

Los ejemplos de codigo tienen borde gris:

```
[EJEMPLO]
Codigo o concepto de ejemplo...
```

### Examenes

La informacion de examenes tiene borde rojo:

```
[EXAMEN]
Titulo del examen
Fecha, lugar, duracion...
```

### Proyectos

Los proyectos tienen borde amarillo:

```
[PROYECTO]
Titulo del proyecto
Fecha de entrega...
```

## Uso en Movil

El sitio esta optimizado para dispositivos moviles:
- La navegacion se adapta al tamano de pantalla
- El contenido es legible sin hacer zoom
- Los botones son suficientemente grandes para tocar

## Atajos de Teclado

- `Tab`: Navegar entre elementos interactivos
- `Enter`: Activar links y botones

## Accesibilidad

- Todos los elementos interactivos son accesibles con teclado
- Los colores tienen suficiente contraste
- Las imagenes tienen texto alternativo
- La fuente OpenDyslexic esta disponible

## Problemas Comunes

### El sitio no carga

1. Verifica tu conexion a internet
2. Intenta recargar la pagina (Ctrl+F5)
3. Limpia la cache del navegador

### El tema no cambia

1. Asegurate de tener JavaScript habilitado
2. Intenta en modo incognito
3. Limpia localStorage del navegador

### La fuente no cambia

1. Espera unos segundos (la fuente se carga desde CDN)
2. Verifica tu conexion a internet
